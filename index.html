cowDuck iPhone App

iPhone


An application made by Wheaton students for Wheaton students. Gives up to date information on the campus dining hall menus, the student/faculty directory, events calendar, course schedule and transportation schedules. cowDuck is a third place winner at CCSCNE (Consortium for Computing Sciences in Colleges - Northeastern Region)

Rubiks Cube Simulator

Mac, Windows


An application that helps teach others how to solve a Rubik's Cube. The app integrates the Leap Motion which allows the user to simulate the movements of a cube with their hands for a more natural feel.

A command line tool that generates python files. It supports class, method/function and Flask application generation. It will install other libraries/modules if user wants to import it in their project.
